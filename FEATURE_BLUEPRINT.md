# KIDORA Quest Feature Blueprint

## Learning Model
Hear → Move → Say → Read → Create → Share

## Integrated Main Areas
1. SoundQuest AI: listening, story clue, target word, command, play/stop audio.
2. EchoMirror AI: speaking, recording, playback, voice evidence.
3. GlyphPath AI: phonics, word cards, sentence path, read-aloud.
4. StoryForge AI: tracing/drawing canvas, sentence train, storybook artifact.
5. Puzzle Bridge: integrated four-skill mission.
6. Living Storybook: multimodal portfolio.
7. Teacher Quest Cockpit: reports and analytics.
8. Admin Control: full app management.

## Built-in Themes
Colours, Animals, Food, Family, Body, Weather, Classroom, Actions.

## Data Evidence
The application saves listening, speaking, reading, writing, drawing, and integrated evidence in browser localStorage and exports CSV.
